# 串流轉檔與播放

* 開啟 RtspToM3u8.bat
* 設定 [攝影機RTSP連結]
* 執行 RtspToM3u8.bat
* 使用 VISual Studio Code  開啟 Play.html
* 使用  執行Go Live 即可瀏覽串流

# 串流轉單張影像

* 開啟 RtspToImageHistory.bat
* 設定 [攝影機RTSP連結]
* 設定 [輸出的資料夾]
* 執行 RtspToImageHistory.bat
* 10秒鐘快取一張圖片

# 串流轉錄影檔

* 開啟 RtspToMp4History
* 設定 [攝影機RTSP連結]
* 設定 [輸出的資料夾]
* 執行 RtspToMp4History
* 五分鐘一個檔案
